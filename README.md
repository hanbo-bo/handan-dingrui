## 关于我们

邯郸鼎锐金属制品有限公司位于河北省邯郸市，是一家专注于出口贸易和紧固件供应的现代化企业。公司致力于将中国金属制品制造资源对接全球市场，为海外客户提供一站式的紧固件采购和定制金属制品出口服务。

### 主营方向

- **出口贸易**：为全球进口商、经销商、工程承包商和工业采购商提供出口服务
- **紧固件供应**：涵盖螺栓、螺母、螺钉、垫圈、锚固件、牙条、不锈钢紧固件及非标金属件
- **定制加工**：按图生产、冲压、铸造、表面处理和品牌化包装
- **供应链服务**：报价、样品确认、生产跟进、质量检验、出口包装、物流协调和长期供应链合作

### 材质与表面处理

- 不锈钢：201 / 304 / 316 / 410 系列
- 碳钢与合金钢：8.8 / 10.9 / 12.9 级高强度
- 铜及黄铜制品
- 表面处理：镀锌、热镀锌、发黑、达克罗、本色等

### 服务优势

- **专业团队**：经验丰富的销售与跟单人员，支持图纸、标准、材质和包装要求确认
- **质量控制**：检验覆盖尺寸、螺纹、材质、硬度、镀层、外观、数量、标签和纸箱
- **价格优势**：工厂直采与品类整合，帮助客户对比更合适的生产方案
- **准时交付**：围绕出口交期管理生产排期、包装跟进和物流协调
- **贸易条款**：支持 FOB、CIF、DDP 及集装箱运输

### 采购流程

1. **询盘** — 提供产品详情、图纸、样品、目标价格和预计数量
2. **报价** — 确认材质、标准、表面处理、起订量、包装和贸易条款
3. **样品** — 批量生产前安排样品或产前确认
4. **生产** — 跟进生产进度，并在出货前进行检验
5. **交付** — 支持出口包装、唛头、单证和物流安排

### 部署方式

推荐使用 **GitHub Pages** 免费部署：

1. 将代码推送到 GitHub 仓库
2. 进入仓库 Settings → Pages
3. Source 选择 Deploy from a branch，Branch 选择 main / root
4. 等待几秒后即可通过 `https://你的用户名.github.io/仓库名` 访问
5. test auto deploy
